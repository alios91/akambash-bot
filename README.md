# AkambashBot

Учебный Telegram-бот для изучения абхазского языка.  
Поддержка русского, английского и турецкого интерфейса.  

## Как запустить:

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python bot.py
```

Бот будет отвечать на команды /start и /word.
